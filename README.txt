●サンプルデータについて
サンプルデータには『Pythonによるクローラー&スクレイピング入門　設計・開発から収集データの解析・運用まで』の本文中で解説した第4章から第9章、Appendixのリストのサンプル（実行ファイル、参考出力ログ）を収録しています。

●サンプルフォルダのディレクトリについて
　サンプルは各リストの番号を元にフォルダで分類しています。

・紙面でファイル名が明記されている場合は、そのファイル名で収録しています。

------------------------------------------
例：Chapter 4のリスト4-3のサンプルの場合
------------------------------------------
「LIST_4-3」フォルダ内にリスト4-3のサンプル「Chapter4-1.py」があります。
------------------------------------------

・紙面で特にファイル名が明記されてない場合は、リスト_（章番号）-（連番）.（関連する拡張子）、で収録しています（一部、本文記載のフォルダ単位で収録しているものもあります）。

------------------------------------------
例：Chapter 4のリスト4-1のサンプルの場合
------------------------------------------
「LIST_4-1」フォルダ内にリスト4-1のサンプル「LIST_4-1.html」があります。
------------------------------------------
